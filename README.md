# Система автоматизации ухода за растениями в ботаническом саду
Клиент-серверное приложение для автоматизации ухода за растениями в ботаническом саду

Проект разрабатывается в рамках учебной дисциплины и реализует систему управления растениями, зонами ботанического сада и задачами по уходу за ними.
---
В проекте используется следующая модель ветвления:

- **main** — стабильная версия проекта  
- **dev** — основная ветка разработки  
- **feature/\*** — ветки для реализации отдельных задач
